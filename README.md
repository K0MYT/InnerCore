# InnerCore Mod

**InnerCore** - это мод, добавляющий блок (пластину) с возможностью размещать на нем множество любых блоков в фиксированную сетку 6×6×6 с полным сохранением их функций.

## Особенности

- **Компактное хранилище**: Разместите до 216 блоков в одной пластине
- **Полная функциональность**: Все блоки сохраняют свои функции (печи, сундуки, редстоун и т.д.)
- **Легкий крафт**: 8 железных слитков + 1 стеклянная панель
- **Интуитивное управление**:
  - **ПКМ с блоком**: Разместить блок
  - **ПКМ пустой рукой**: Открыть содержимое
  - **Shift + ЛКМ**: Забрать блок обратно
  - **Shift + ПКМ**: Разместить на интерактивных блоках

## Применение

Идеален для создания компактных сложных схем с помощью модов вроде:
- **Create Aeronautics** - создание транспорта
- **Create Mod** - механизмы
- **ComputerCraft: Tweaked** - программирование

## Версия

- Minecraft: 1.21.1
- NeoForge: 21.1.228

## Установка

1. Установите NeoForge 21.1.228
2. Поместите JAR файл мода в папку `mods`
3. Запустите игру
